Professional Engineering Projects as a Sr Systems Engineer at Moog

Heterogeneous Distributed Sensor Fusion & Multi-object Tracking

Acted as the technical lead for Moog’s AI Target Recognition, developing a novel sensor fusion algorithm (multiple radar, multi-spectral video) in a docker framework capable of real-time multi-object tracking.

Mid-wave Infrared and Electro-optical Small Object Detection

Developed an automated performance evaluation pipeline against several statistical metrics for object detection CNN models using PyTorch in the electro-optical and medium wave infrared vision spectrums.

Excavator Autonomy - Basic Research Collaboration

Facilitated a collaboration between Moog and the University at Buffalo to develop an autonomy stack for a pair of 6-ton excavators where I mentored and coached a team of 8 graduate students.

Production Space Vehicles GNC Integration & Test

Contributed to several of Moog’s earliest programs in the space vehicles business given a challenging schedule for the GNC integration and test team, utilizing C++ and pytest for hardware in the loop unit testing and requirement verification.

Advanced Precision Kill Weapon System (APKWS) Integration

Led the integration of the Advanced Precision Kill Weapon System (APKWS) rocket into Moog’s stores management software, using modern C++ and CMake on embedded Linux (Ubuntu, RHEL) devices.

Autonomous Material Handling - Off-highway Perception

Architected a multi-object detection and tracking algorithm utilizing C++ and ROS2 that uses lidar sensor data and enables collision avoidance for fully autonomous 6-ton Bobcat T870 construction vehicles. Worked with the autonomy safety team verifying perception sensors and systems adhere to all necessary standards, such as ISO 26262 (functional safety) and ISO 21448 (safety of the intended functionality).

Reinforcement Learning Excavator Control

Implemented an embedded reinforcement learning controller using python, C++, and ROS2 capable of controlling the end effector of a hydraulic excavator arm using task-space commands.
